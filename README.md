# Storm Chaser

## Overview

Storm Chaser is a realtime weather App that shows an extendted 5 day forcast.

## Prerequites

1. clone repository- https://github.com/DavidSwider/Strom-Chaser.git
2. npm install
3. configure your environment folder
4. run build
5. run start

## Tech

- TypeScript
- HTML
- CSS
- Node
- Express
- Weather API
